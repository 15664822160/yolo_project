##基于Yolov8的摔倒检测系统##
功能：检测小人模型摔倒或站立，判断为摔倒则发送摔倒的视频至微信紧急联系人。
摔倒判断条件：视频以每秒30帧运行，系统会记录最近200帧图像，当200帧图像中有60%判断为摔倒时，则将200帧图片合成视频发送至微信文件传输助手，此功能只有视频文件按钮能使用
runs文件：模型训练后的文件，训练了2个模型，fall\runs\detect\train为从互联网上找到的数据集素材，可以识别人体摔倒无法识别站立
fall\runs\detect\train2为自己拍摄的模型小人作为数据集，默认采用次模型，可以识别小人站立或摔倒
my_dataset文件：包含训练集和验证集的图片及标注
main.py：主程序，引入许多库，需配置好后运行使用，下代码复制粘贴至Pycharm终端，python解释器最好在3.8以上
pip install PyQt5
pip install ultralytics
pip install wxauto
pip install opencv-python
Yolov8n.pt：预训练模型
test.mp4：从程序中选择视频文件中选择此视频测试，注意需提前登录微信，否则导致发送信息错误（可能是我把小人头摔掉的原因，导致站立检测概率低）
expectmp4：此视频为操作视频，也是预期结果演示视频。
注：此系统仅供学习，娱乐，不得作为商业用途
